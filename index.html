Vítejte na GitHub stránek. Tato automatická stránka generátor je nejjednodušší způsob, jak vytvořit krásné stránky pro všechny vaše projekty. Autor svůj obsah stránky Here pomocí GitHub ochucené Markdown, vyberte šablonu vytvořený návrhářem a publikovat. Po vygenerování vaše stránky, můžete vyzkoušet novou gh-pages větev lokálně. Pokud používáte GitHub Desktop, jednoduše synchronizovat úložiště a uvidíte novou pobočku. ### Šablony Designer jsme řemeslně nějaké hezké šablony pro vás k použití. Pokračujte a klikněte na tlačítko "Pokračovat rozvržení" procházet skrze ně. Můžete snadno vrátit do upravovat své stránky před publikováním. Po publikování svou stránku, můžete vrátit na stránky generátor a přejít k jinému tématu. Svůj obsah stránky budou zachovány. ### Vytváření stránek ručně Chcete-li raději nepoužívat automatický generátor, tlačit pobočku s názvem gh-pages do vašeho úložiště ručně vytvořit stránku. Kromě podpory pravidelné HTML obsah, GitHub Strany podporují Jekyll, jednoduchý, blog vědomi statický generátor webu. Jekyll usnadňuje vytvořit celý web záhlaví a zápatí bez nutnosti jejich kopírování přes každé stránce. Nabízí také inteligentní podporu blogu a další pokročilé funkce šablon. ### Autorů a spolupracovníků Můžete @mention uživatelské jméno GitHub vygenerovat odkaz na jejich profilu.
